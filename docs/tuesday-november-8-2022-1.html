  1. Use the us_counties() function from the {USAboundaries} package to get county boundaries in Florida as a simple feature data frame called Bay.sf. Filter on name to keep only Bay County. Transform the geometry using EPSG 3086. (5)
Bay.sf <- USAboundaries::us_counties(states = "FL") |>
  dplyr::select(name) |>
  dplyr::filter(name == "Bay") |>
  sf::st_transform(crs = 3086)
  1. Use the function st_intersection() to create a new simple feature data frame called BayFires.sf from the FL_Fires.sf simple feature data frame that contains only the lightning caused wildfires inside the Bay County polygon. (10)
BayFires.sf <- FL_Fires.sf |>
  sf::st_intersection(Bay.sf) |>
  dplyr::filter(STAT_CAU_1 == "Lightning")
